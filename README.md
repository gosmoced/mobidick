### ⚡️ Key Features

This project was built to provide a fast, accessible, and user-friendly entry point for agency clients.
* **Multi-Language Support:** Integrated language switcher allowing users to easily toggle between languages for a diverse target audience.
* **Fully Responsive:** Looks great and functions perfectly on all devices, from mobile phones to 4K desktops.
* **Modern Styling:** Built using **Tailwind CSS** for a sleek, consistent, and highly optimized user interface.
* **Lightweight & Fast:** Minimalistic approach with no heavy frameworks, ensuring quick load times and smooth performance.

### 🛠 Tech Stack

* **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript.
* **Tools:** Git, PyCharm / VS Code.
* **Deploy:** Vercel.

### 🚀 How it works

* The site uses a lightweight, straightforward architecture to ensure maximum speed.
* **Tailwind CSS** handles the entirely responsive layout and utility-first styling without the need for custom CSS files.
* **Vanilla JavaScript** drives the language switching functionality and basic UI interactions.
* Deployed seamlessly via Vercel with automatic updates on push.
